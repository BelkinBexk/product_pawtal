3. Critical — App Store Payment Cut
This directly impacts Pawtal's commission model on mobile
⚠️ Apple and Google charge 15–30% on all in-app payments. This fee is on top of your payment gateway fee and would severely compress Pawtal's per-booking margin if payments are collected inside the native app.
Payment Route Apple/Google Cut Gateway Fee Pawtal Net (฿488 booking) Recommended?
In-App Payment (native) 15–30% 3.65% Omise ฿11–33 net ✗ Avoid
WebView / Browser Redirect ฿0 3.65% Omise ฿55 net ✓ Recommended
Web App (PWA / Next.js) ฿0 3.65% Omise ฿55 net ✓ Recommended
Solution: When user taps "Pay" in the mobile app, redirect to Omise Checkout in the device browser. Payment completes → user is redirected back to the app. This is a fully supported Omise pattern and completely avoids the Apple/Google cut.
